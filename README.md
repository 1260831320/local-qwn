# 鸿栀 v1.0.5

鸿栀是一套本地优先的智能工作台。

当前版本把重点放在四件事上：

- 用中文优先的界面完成阅读、对话和状态查看
- 在普通电脑上保持可用的本地体验
- 让重要变更先预览、后确认
- 对未开放的能力保持明确说明，不假装已经实现

## 你可以做什么

- 阅读当前版本说明，并按章节快速定位内容
- 在工作台中直接提问，查看回答和处理过程
- 根据需要切换回答引擎与回答模式
- 在确认后应用变更，并查看最近记录

## 三个主要页面

### 总览

- 快速了解当前版本能做什么
- 用更直观的流程卡片查看推荐使用顺序
- 了解不同使用模式的特点

### 工作台

- 优先展示当前对话
- 在同一页查看服务状态与使用偏好
- 遇到变更时先看预览，再决定是否应用

### 手册

- 支持中文与 English 双语阅读
- 支持章节摘要、目录定位和阅读进度
- 看完可以直接带着问题回到工作台

## 使用方式

1. 启动应用。

Windows:

```powershell
.\mvnw.cmd spring-boot:run
```

macOS / Linux:

```bash
./mvnw spring-boot:run
```

2. 在浏览器打开 `http://localhost:8080`
3. 先看“总览”或“手册”，再进入“工作台”
4. 如需应用变更，请先阅读预览内容

## 当前版本说明

- 版本：`v1.0.5`
- 形态：本地优先智能工作台
- 重点：核心功能优先展示、用户口径文案、双语手册阅读

## 本地体验原则

- 本地优先：优先使用本地服务与本地推理
- 隐私友好：不依赖云端必需能力
- 清晰可见：重要状态和当前模式直接展示
- 安全确认：涉及写入时先预览、再确认

## 可用引擎

当前版本提供两种本地回答引擎：

- `Ollama`
- `OpenVINO`

一般建议：

- 问题更复杂、希望回答更完整时，优先使用 `Ollama`
- 更看重轻量运行和资源占用时，可以尝试 `OpenVINO`

## 当前暂未开放

以下区域仍是预留位，当前版本不会伪造能力：

- 语音模式
- 麦克风 / 摄像头连接
- 数字形象表达
- 情绪建议与干预结果

## v1.0.5 更新

- 首页、工作台、手册改为核心内容优先展示
- 左侧常驻区支持折叠，给主要内容让出更多空间
- 全站前端文案改为用户口径，移除开发者表达
- 更新首页与浏览器图标为新 logo
- 双语手册同步更新为当前版本说明

## 提醒

- 当前版本仍在持续完善中
- 如果某项服务暂时不可用，界面会直接提示当前状态
- 在变更确认前，不会直接写入文件
